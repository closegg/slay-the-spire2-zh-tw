### 🔧 使用說明
1. 下載 `STS2繁體漢化安裝器.exe`。
2. 確保資料夾內包含「slay the spire2 繁體漢化」資料夾。
3. 執行安裝器，點擊「開始安裝」。
4. 進遊戲後在設定中切換為「繁體中文」。

### ⚠️ 注意事項
* 執行時若出現 Windows 保護提醒，請點擊「其他資訊」並選擇「仍要執行」。
